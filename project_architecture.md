# Her Echoes - Project Architecture

## Overview
Her Echoes is a Flutter-based emotional journaling and reflection app.

## Tech Stack
- Flutter (Frontend)
- Dart
- Local Database (Excel → future SQLite)
- GitHub for version control
- Apple Developer Account (future iOS release)
